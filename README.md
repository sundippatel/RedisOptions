## Installation


- clone via Composer (recommended):

-- add to app's composer.json:
 
```json
"require": {
    "sunra/redis-options":"dev-master"
    },
"repositories": [
        {
            "type": "git",
            "url": "https://github.com/sunra/RedisOptions"
        }
    ],
```
